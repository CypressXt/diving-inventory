# Diving inventory

This project aim to keep on tracking diving tanks lifecycle.

## Features
  - ✅ Mobile friendly
  - ✅ Diving tank inventory
  - ✅ Diving tank pressure management
    - ✅ At inflation
    - ✅ After a dive
  - ✅ Dive log by tank
  - ✅ Diving tank oxygen mixture (Nitrox o2%)
  - ✅ Diving tank identification by QRcode
  - [ ] User management
  - [ ] User admin role
  - [ ] i18n multilingual
